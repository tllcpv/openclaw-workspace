## 二、选股策略（分周期）

### 2.1 短期策略（持股1-5天）
_目标：博弈连板龙头的第二波机会_

#### 硬性门槛
| 指标 | 标准 | 说明 |
|------|------|------|
| 市值 | >20亿 | 避免流动性风险 |
| 历史连板 | 首板或二板 | 必须走过一波连板 |
| 回调幅度 | <20% | 不能跌太深 |
| 题材持续性 | 主线题材 | 非一日游概念 |
| 其他 | 成交量不能萎缩 | 保持活跃 |

#### 核心选股逻辑
**选股范围：**
- 近期有过首板或二连板的股票
- 处于热点主线题材（非边缘题材）

**买入时机：**
- 连板断板后出现回调
- **回调时间：2-3天观察期**
- **回调幅度：不大（<20%），缩量回调**
- **不跌破关键支撑（5日/10日线）**
- 成交量明显萎缩（表明抛压释放完毕）
- **成交量萎缩后再度放量**（资金回流信号）
- 出现止跌信号（十字星、小阳线）
- **题材是主线，非一日游**
- **结合当时短期情绪判断是否有二波预期**

**确认二波的信号：**
- 早盘快速拉升突破前高
- 板块内有其他标的跟风涨停
- 市场情绪转暖（涨停家数增加）

- [x] ST/退市风险
- [ ] 三连板以上后回调（位置太高）
- [ ] 回调跌破20日线或跌停
- [ ] 题材已退潮（无跟风标的）
- [ ] 大盘情绪冰点（涨停家数<20家）
- [ ] 其他：_________

### 2.2 中期策略（持股1-4周）
_目标：上涨趋势中，区间下限博反弹_

#### 硬性门槛
| 指标 | 标准 | 说明 |
|------|------|------|
| 趋势 | 明确上升趋势 | 均线多头排列 |
| 震荡区间 | 已形成 | 有清晰的上下轨 |
| 回调位置 | 区间下限附近 | 不能跌破趋势 |
| 成交量 | 回调缩量 | 表明调整健康 |
| 其他 | 板块处于热点期 | 有持续性 |

#### 核心选股逻辑
**选股范围：**
- 处于明确上升趋势的股票
- 已形成清晰的震荡整理区间
- 板块处于市场主线，有持续性

**买入时机：**
- 股价走到震荡区间的**下限位置**
- **回调缩量**（健康调整的信号）
- 不跌破上升趋势线或重要均线（如20日线）
- 出现止跌形态（下影线、缩量十字星）
- 板块内其他标的未出现集体走弱

**目标位：**
- 区间上限（震荡上沿）
- 若突破区间，转趋势持有

**持有周期：**
- 一般1-4周
- 到达区间上限或出现放量滞涨时考虑离场

#### 一票否决
- [ ] 跌破震荡区间下限（趋势可能改变）
- [ ] 放量下跌（资金出逃）
- [ ] 板块集体退潮
- [ ] 大盘系统性风险
- [ ] 其他：_________

### 2.3 长期策略（持股1个月以上）
_目标：板块困境反转，或长期热点趋势持有_

#### 硬性门槛
| 指标 | 标准 | 说明 |
|------|------|------|
| 行业周期 | 长期低迷后筑底/反转迹象 | 或高景气持续性热点 |
| 板块逻辑 | 清晰的反转催化剂 | 政策/供需/技术突破 |
| 个股地位 | 行业龙头/核心标的 | 有弹性 |
| 估值 | 相对合理 | 困境股低估值，热点股看空间 |
| 业绩 | 底部改善/高增预期 | 验证逻辑 |

#### 核心选股逻辑
**选股类型一：困境反转**
- 行业经历长期低迷，估值处于历史低位
- 出现明确的反转催化剂（政策、供需、技术）
- 板块内优质标的先行启动
- 业绩开始底部改善

**选股类型二：长期热点趋势**
- 有长期持续性的热点板块（如AI、新能源革命）
- 板块处于高景气周期
- 个股是产业链核心环节
- 业绩持续高增长或预期强烈

**买入时机：**
- **困境反转型：** 板块筑底完成，龙头率先启动时
- **热点趋势型：** 回调到重要均线（如30日/60日线），趋势未破坏
- 成交量温和放大（资金开始关注）
- 市场整体情绪不极端

**持有逻辑：**
- 困境反转：持有至估值修复完成或业绩兑现
- 热点趋势：沿着趋势持有，不破关键均线不出

**卖出条件：**
- 困境反转：估值回到合理区间/业绩增速放缓
- 热点趋势：趋势破坏/板块退潮/出现更好标的

#### 一票否决
- [ ] 反转逻辑证伪
- [ ] 行业出现新的长期利空
- [ ] 个股基本面恶化
- [ ] 估值泡沫化（热点股）
- [ ] 其他：_________
