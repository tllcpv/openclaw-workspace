# 股票研究笔记

> 记录观察、分析与操作。记忆比预测更可靠。

---

## 📁 文件结构

| 文件 | 用途 |
|------|------|
| `watchlist.md` | 当前观察的股票清单及关注点 |
| `trades.md` | 所有操作记录（买入/卖出/持仓变动） |
| `notes/个股代码.md` | 单只股票的详细跟踪笔记 |

---

## 🎯 使用方式

### 添加新股票到观察清单
在 `watchlist.md` 中添加，注明关注原因和关键价位。

### 创建个股跟踪笔记
在 `notes/` 下新建文件，命名格式：`代码-简称.md`（如 `000001-平安银行.md`）

### 记录操作
在 `trades.md` 中按日期倒序记录，包含：日期、操作、价格、数量、理由。

### 定期更新
- 重大新闻/财报后更新个股笔记
- 每周回顾 watchlist，移除不再关注的标的
- 月底复盘 trades，总结得失
