# 评估资料字段定义

评估项目基础资料的标准字段体系，用于指导资料接收和解析。

## 一、基本信息 (basic_info)

| 字段名 | 说明 | 常见表述 | 是否必填 |
|--------|------|----------|----------|
| property_name | 项目名称 | 项目名称、房产名称、标的物名称、估价对象 | 是 |
| property_address | 项目地址 | 项目地址、房产地址、地址、坐落 | 是 |
| property_type | 房产类型 | 房产类型、房屋类型、用途、物业类型 | 是 |
| owner | 产权人 | 产权人、所有人、权利人、业主 | 是 |

## 二、物理状况 (physical_info)

| 字段名 | 说明 | 常见表述 | 是否必填 |
|--------|------|----------|----------|
| building_area | 建筑面积 | 建筑面积、面积、总面积 | 是 |
| land_area | 土地面积 | 土地面积、占地面积 | 否 |
| floor_count | 总层数 | 层数、总层数、楼层 | 否 |
| current_floor | 所在楼层 | 所在楼层、层 | 否 |
| build_year | 建成年份 | 建成年份、竣工年份、建成年代 | 是 |
| structure | 建筑结构 | 建筑结构、结构类型、结构 | 否 |

## 三、权属信息 (rights_info)

| 字段名 | 说明 | 常见表述 | 是否必填 |
|--------|------|----------|----------|
| certificate_no | 产权证号 | 产权证号、不动产权证号、房产证号 | 是 |
| land_certificate | 土地证号 | 土地证号、土地使用权证号 | 否 |
| land_use_type | 土地用途 | 土地用途、用地性质 | 否 |
| land_use_period | 使用期限 | 土地使用期限、使用权期限、终止日期 | 否 |

## 四、租赁信息 (rental_info)

| 字段名 | 说明 | 常见表述 | 是否必填 |
|--------|------|----------|----------|
| current_rent | 当前租金 | 当前租金、现行租金、月租金 | 否 |
| lease_start | 租赁起始日 | 租赁起始日、起租日期 | 否 |
| lease_end | 租赁到期日 | 租赁到期日、到期日期 | 否 |
| tenant | 承租人 | 承租人、租户、租客 | 否 |

## 五、资料完整性检查清单

### 基础资料（必须）
- [ ] 产权证书扫描件
- [ ] 身份证明文件
- [ ] 房产平面图/位置图
- [ ] 现场照片（外观、内部、周边环境）

### 评估特定资料
**租金评估：**
- [ ] 租赁合同
- [ ] 历史租金收入凭证
- [ ] 同类物业租金调查表

**转让价格评估：**
- [ ] 购房合同/发票
- [ ] 近期同类成交案例

**抵押评估：**
- [ ] 借款合同
- [ ] 抵押物清单
- [ ] 评估委托书

## 六、字段提取规则

### 数值类字段
- 建筑面积：提取数字+单位，统一转换为"平方米"
- 金额：提取数字+货币单位，统一转换为"万元"或"元"
- 日期：统一格式为 YYYY-MM-DD

### 文本类字段
- 去除前后空格
- 统一全角半角标点
- 去除换行符

### 地址类字段
- 保留完整行政区划路径
- 去除"详细地址："等前缀
