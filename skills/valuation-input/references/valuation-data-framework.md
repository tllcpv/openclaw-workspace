# 评估项目通用信息框架

评估项目基础资料的通用分类标准，适用于各类评估项目（抵押、转让、征收、司法鉴定等）。

## 一、抵押物基本信息

| 字段名 | 说明 | 数据类型 | 示例 |
|--------|------|----------|------|
| project_name | 项目名称 | 文本 | XX公司抵押评估 |
| project_type | 项目类型 | 单选 | 抵押/转让/征收/咨询/司法鉴定 |
| asset_type | 资产类型 | 单选 | 住宅/商业/办公/工业/土地/在建工程 |
| asset_name | 资产名称 | 文本 | XX大厦/XX厂房/XX地块 |
| asset_location | 资产位置 | 文本 | XX省XX市XX区XX路XX号 |
| debtor_name | 债务人名称 | 文本 | XX公司/XX个人 |
| creditor_name | 债权人名称 | 文本 | XX银行/XX机构 |
| loan_amount | 担保金额 | 数值 | 1000万元 |

## 二、抵押物权属信息

| 字段名 | 说明 | 数据类型 | 示例 |
|--------|------|----------|------|
| owner | 产权人 | 文本 | XX公司 |
| certificate_no | 产权证号 | 文本 | XX房权证XX字第XXXX号 |
| land_certificate | 土地证号 | 文本 | XX国用(20XX)第XXXX号 |
| land_nature | 土地性质 | 单选 | 出让/划拨/集体/租赁 |
| land_use_type | 土地用途 | 单选 | 住宅/商服/工业/综合/其他 |
| use_period_start | 使用期限起始日 | 日期 | 20XX-XX-XX |
| use_period_end | 使用期限终止日 | 日期 | 20XX-XX-XX |
| ownership_type | 共有情况 | 单选 | 单独所有/共同共有/按份共有 |
| other_rights | 他项权利 | 多选 | 抵押/查封/异议/预告登记 |

## 四、评估测算信息

### 4.1 方法选择

| 字段名 | 说明 | 数据类型 |
|--------|------|----------|
| valuation_method | 评估方法 | 多选 | 市场法/收益法/成本法/假设开发法 |
| method_reason | 方法选择理由 | 文本 | 根据资产类型和评估目的选用 |

| 字段名 | 说明 | 数据类型 | 示例 |
|--------|------|----------|------|

### 4.3 市场法参数

### 4.7 评估限制条件

| 字段名 | 说明 | 数据类型 |
|--------|------|----------|

| value_date | 价值时点 | 日期 |
| value_type | 价值类型 | 市场价值/抵押价值/投资价值等 |
| assumptions | 假设条件 | 文本 |
| limitations | 限制条件 | 文本 |

## 五、信息关联关系

```
基本信息（是什么）
    ↓
权属信息（合法吗）
    ├─ 产权清晰 → 继续
    └─ 权属瑕疵 → 风险提示
    ↓
现状信息（什么样）
    ├─ 现场核实 → 照片/视频/记录
    └─ 使用状况 → 收益法依据
    ↓
测算信息（值多少）
    ├─ 方法选择 → 适用性分析
    ├─ 参数收集 → 市场调查/造价指标
    ├─ 计算过程 → 公式与逻辑
    ├─ 结果校验 → 横向对比/经验判断
    └─ 结论确定 → 单价→总价→变现价值
```

## 六、不同资产类型的必填字段

### 住宅类
- 建筑面积、楼层、朝向、建成年代、装修状况

### 商业类
- 建筑面积、楼层、临街状况、客观租金、收益期限

### 办公类
- 建筑面积、楼层、装修状况、物业管理、配套设施

### 工业类
- 建筑面积、土地面积、建筑结构、建成年代、容积率

### 土地类
- 土地面积、土地用途、使用期限、规划条件、开发程度

## 七、数据质量要求

| 等级 | 要求 |
|------|------|
| 必填 | 缺少则无法完成评估（权属信息、基本面积） |
| 建议 | 影响评估精度（详细区位、维护状况） |
| 可选 | 锦上添花（特殊装修、设备设施） |

---

*此框架适用于各类房地产评估项目，可根据具体项目类型和委托方要求调整字段。*
